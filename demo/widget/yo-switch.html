假设你想要其他颜色，那么你可以这样扩展：

命名及颜色按照各自团队内部的FE/UI规范即可

全局扩展方式（位置参数）：

@include yo-switch(test,#4a87ee,#4a87ee);

全局扩展方式（关键字参数）：

@include yo-switch( $name: test, $checked-bordercolor: #4a87ee, $checked-bgcolor: #4a87ee );

使用方法：

<label class="yo-switch yo-switch-test"> <input type="checkbox" /> <div class="track"> <span class="handle"></span> </div> </label>

局部扩展方式（位置参数）：

.xxx{ @include yo-switch(test,#4a87ee,#4a87ee); }

全局扩展方式（关键字参数）：

.xxx{ @include yo-switch( $name: test, $checked-bordercolor: #4a87ee, $checked-bgcolor: #4a87ee ); }

使用方法：

<div class="xxx"> <label class="yo-switch yo-switch-test"> <input type="checkbox" /> <div class="track"> <span class="handle"></span> </div> </label> </div> 
@include yo-switch(
    $name:default, //{default | String} switch名称
    $checked-bordercolor:default, //{default | Color} 定义switch激活边框色
    $checked-bgcolor:default, //{default | Color} 定义switch激活背景色
    $bordercolor:default, //{default | Color} 定义switch边框色
    $bgcolor:default, //{default | Color} 定义switch背景色
    $ball-color:default //{default | Color} 定义switch滑块色
)
