假设你想要背景色，那么你可以这样扩展：

命名及颜色按照各自团队内部的FE/UI规范即可

全局扩展方式：

@include yo-loading( $name: b, $bgcolor: #666, $mask-bgcolor: rgba(0, 0, 0, .2), $ico-color: #00afc7, $color: #00afc7 );

使用方法：

<div class="yo-loading yo-loading-b"> <span class="inner"><i class="yo-ico"></i>加载中...</span> </div>

局部扩展方式：

.xxx{ @include yo-loading( $name: b, $bgcolor: #666, $mask-bgcolor: rgba(0, 0, 0, .2), $ico-color: #00afc7, $color: #00afc7 ); }

使用方法：

<div class="xxx"> <div class="yo-loading yo-loading-b"> <div class="inner"><i class="yo-ico"></i>加载中...</div> </div> </div> 
@include yo-loading(
    $name:default, //{String} yo-loading名称
    $ico-size:default, //{Length} 菊花大小
    $ico-color:default, //{Color} 菊花颜色
    $mask-bgcolor:default, //{Color} 遮罩颜色
    $bgcolor:default, //{Color} 背景颜色
    $font-size:default, //{Length} 文本字号
    $color:default, //{Color} 文本颜色
    $content:default //{String} loading形态，可传入webfonts编码
)
