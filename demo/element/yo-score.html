Score

常规：

打分：

当前评分：3分

扩展：

打分：

当前评分：3分

<div class="yo-score"> <h3 class="index" style="width:60%;">当前评分：3分</h3> </div>

假设你想要其他风格，那么你可以这样扩展：

命名及颜色按照各自团队内部的FE/UI规范即可

全局扩展方式：

@include yo-score( $name: test, $url: "star2.png" );

使用方法：

<div class="yo-score yo-score-test"> <h3 class="index" style="width:60%;">当前评分：3分</h3> </div>

局部扩展方式：

.xxx { @include yo-score( $name: test, $url: "star2.png" ); }

使用方法：

<div class="xxx"> <div class="yo-score yo-score-test"> <h3 class="index" style="width:60%;">当前评分：3分</h3> </div> </div> 
@include yo-score(
    $name:default, //{String} yo-score名称
    $item-width:default, //{Length} 单项宽度
    $item-height:default, //{Length} 单项高度
    $url:default //{String} 图片URL
)
