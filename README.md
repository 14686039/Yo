<a name="version-intro"></a>
### 版本号说明

Yo 遵循 [Semantic Versioning](http://semver.org/lang/zh-CN/) 规范，版本采用 `主版本号.次版本号.修订号` 的格式，规则如下：

* 主版本号：做了不兼容的API修改，比如大规模重构，设计模式变更，语法变更等；
* 次版本号：做了向下兼容的功能性新增；
* 修订号：做了向下兼容的问题修正、细节调整等。
